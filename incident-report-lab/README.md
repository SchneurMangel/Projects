# Incident Report Lab

**Google Cybersecurity Certificate | Portfolio Project**  
**Author:** Schneur Mangel  

---

## Overview

This project is an incident handler's journal entry documenting a brute force attack investigation. It covers how to record what happened during a security incident in a clear and structured way.

This report is based on findings from two other labs in this portfolio — the packet analysis lab and the SIEM investigation lab. Both found evidence of the same attack from different angles.

---

## What this lab covers

- Writing up an incident using the 5 W's format (Who, What, Where, When, Why)
- Documenting tools used and what they found
- Recording response actions taken
- Making recommendations to prevent the same attack happening again
