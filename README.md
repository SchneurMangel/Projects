# Cybersecurity Projects

A collection of hands on labs completed as part of the Google Cybersecurity Certificate and CompTia Security+. Each project covers a core skill area and documents what I learned along the way.

---

## Labs

### [Packet Analysis Lab](./packet-analysis-lab)
Investigated a network intrusion using tcpdump and Wireshark. Identified C2 traffic on port 4444, data exfiltration, an internal port scan, and suspicious DNS queries. Includes a Python script that applies the same detection logic.

### [SIEM Investigation Lab](./siem-lab)
Used Chronicle SIEM to investigate a brute force attack. Wrote UDM queries to confirm the attack, identify the compromised account, and check what the attacker did after getting in.

### [Incident Report Lab](./incident-report-lab)
Documented the brute force investigation as a formal incident handler's journal entry using the 5 W's format. Covers findings, response actions, and recommendations.
